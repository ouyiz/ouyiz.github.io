# 一.本节代码
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>HTML 列表示例</title>
</head>
<body>

  <h2>1️⃣ 无序列表（ul）</h2>
  <p>喜欢的水果：</p>
  <ul>
    <li>苹果</li>
    <li>香蕉</li>
    <li>橘子</li>
  </ul>

  <h2>2️⃣ 有序列表（ol）</h2>
  <p>制作蛋糕的步骤：</p>
  <ol>

    <li>搅拌面糊</li>

    <li>冷却后装饰</li>
  </ol>

  <h2>3️⃣ 有序列表设置属性</h2>

  <ol type="A" start="3">
    <li>项目C</li>
    <li>项目D</li>
    <li>项目E</li>
  </ol>

  </ul>

</body>
</html>

```
# 二. 列表的类型

|列表类型|标签|特点|常用场景|
|---|---|---|---|
|无序列表|`<ul>`|项目前有圆点，顺序不重要|导航栏、分类目录|
|有序列表|`<ol>`|项目前有编号，顺序重要|步骤、排名|
|自定义列表|`<dl>`|术语-解释结构|词汇表、问答FAQ|

# 三. 无序列表（ul）
**结构：**
```html
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橘子</li>
</ul>
```
- `<ul>`：无序列表的外层容器；
- `<li>`：每一项（list item）；
- 默认前面会有小黑点（可用CSS修改 `list-style`）。

# 四. 有序列表（ol）
**结构：**
```html
<ol>
  <li>准备食材</li>
  <li>洗净切好</li>
  <li>开始烹饪</li>
</ol>
```
- `<ol>`：有序列表的外层容器；
- `<li>`：每一个步骤；
- 默认按数字 `1. 2. 3.` 编号。

**常用属性：**

| 属性      | 说明          | 示例                         |
| ------- | ----------- | -------------------------- |
| `type`  | 序号类型（数字、字母） | `<ol type="A">` → A、B、C... |
| `start` | 序号起始值       | `<ol start="5">` 从5开始编号    |

# 五. 自定义列表（dl）

**结构：**
```html
<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言</dd>
  <dt>CSS</dt>
  <dd>层叠样式表</dd>
</dl>
```
- `<dl>`：定义列表容器；
- `<dt>`：定义项（标题，term）；
- `<dd>`：定义描述（definition）。


## 六. 列表嵌套
列表可以**嵌套**使用，常见于菜单或目录：
```html
<ul>
  <li>水果
    <ul>
      <li>苹果</li>
      <li>香蕉</li>
    </ul>
  </li>
  <li>蔬菜</li>
</ul>
```


# 七. 列表注意事项

1.`<li>` 标签只能放在 `<ul>` 或 `<ol>` 内部。  
2️.列表项可以嵌套标签，例如：

```html
<li><a href="#">首页</a></li>
```

3️. `<dl>` 中，`<dt>` 和 `<dd>` 成对出现，适合词典类结构。
